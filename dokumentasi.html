1. Memulai Proyek

Sebelum memulai, pastikan Anda memiliki struktur file dan folder yang benar. Struktur ini penting agar semua aset seperti gambar, CSS, dan JavaScript dapat terhubung dengan benar.

Struktur Folder

/portfolio-project
├── css/

├── js/
│   └── script.js

Daftar File dan Fungsinya

File / Folder Deskripsi
index.html Halaman utama (Home) yang menampilkan foto profil dan nama dengan efek ketik.
about.html Halaman "About Me" yang berisi deskripsi singkat tentang diri Anda.

images/ Folder untuk menyimpan semua aset gambar yang digunakan di website.

2. Struktur Halaman HTML

Semua file HTML (index.html, about.html, projects.html) memiliki struktur dasar yang sama untuk menjaga konsistensi. Mari kita bedah struktur file index.html sebagai contoh.

Struktur Umum

<!DOCTYPE html>
<html lang="en">
<head>
    <!-- Metadata, Judul Halaman, dan Link ke CSS & Font Awesome -->
</head>
<body>
    <div class="page-wrapper">
        <header>
            <!-- Navigasi -->
        </header>

        <main class="container">
            <!-- Konten utama yang berbeda di setiap halaman -->
        </main>

        <footer>
            <!-- Ikon Sosial Media -->
        </footer>
    </div>
    <script src="js/script.js"></script>
</body>
</html>
  • <head>: Berisi metadata, judul halaman, dan tautan ke file style.css serta library Font Awesome untuk ikon.
  • .page-wrapper: Pembungkus utama halaman. Div ini digunakan untuk menerapkan animasi transisi saat halaman dimuat.
  • <header>: Bagian atas halaman yang berisi navigasi. Posisinya fixed sehingga selalu terlihat.
  • <main class="container">: Konten utama halaman. Isinya akan berbeda untuk halaman Home, About, dan Projects.
  • <footer>: Bagian bawah halaman yang berisi ikon sosial media. Posisinya juga fixed.
  • <script>: Memanggil file JavaScript. Hanya diperlukan di index.html untuk efek ketik.
Penting: Navigasi di setiap halaman memiliki link yang ditandai dengan kelas .active. Kelas ini digunakan oleh CSS untuk memberikan garis bawah pada link halaman yang sedang aktif. Pastikan Anda memindahkannya sesuai halaman yang sedang diedit.

3. Penjelasan Kode CSS (style.css)

File style.css adalah pusat kendali visual dari portofolio ini. Kode di dalamnya dibagi menjadi beberapa bagian untuk kemudahan pemeliharaan.

Global Styles & Variables

Bagian paling atas CSS mendefinisikan variabel global. Ini adalah cara termudah untuk mengubah tema situs Anda secara konsisten.

:root {
    --background-color: #FFFFFF;
    --text-color: #111111;
    --font-heading: 'Poppins', sans-serif;
    --font-body: 'Inter', sans-serif;
}

Anda bisa mengubah nilai variabel ini untuk mengganti warna latar, warna teks, dan jenis font di seluruh situs tanpa harus mengubah setiap elemen satu per satu.

Animasi Transisi Halaman

Setiap kali halaman dimuat, akan ada animasi fade in yang halus. Ini dicapai dengan CSS Keyframes.

@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

.page-wrapper {
    animation: fadeIn 0.6s ease-in-out;
}

Animasi fadeIn diterapkan pada .page-wrapper, membuat semua konten muncul secara perlahan dari bawah.

Layout Utama (Flexbox)

Layout utama diatur menggunakan Flexbox pada elemen <main class="container">. Ini memungkinkan konten (misalnya, gambar dan teks di halaman utama) untuk disejajarkan secara horizontal dengan rapi.

main.container {
    display: flex;
    align-items: center;
    justify-content: center;
    gap: 5%;
}

Pada layar kecil (responsif), properti flex-direction diubah menjadi column agar konten tersusun secara vertikal.

Projects Page (Grid Layout)

Halaman proyek menggunakan CSS Grid untuk membuat layout "bento-box".

.projects-grid {
    display: grid;
    grid-template-columns: 1fr 1fr; /* Dua kolom sama besar */
    gap: 20px;
}

.project-box.p1 {
    grid-column: 1 / -1; /* Proyek pertama membentang selebar 2 kolom */
}

Struktur ini fleksibel dan mudah diubah jika Anda ingin menambah atau mengubah tata letak proyek.

Efek Hover pada Proyek

Saat kursor diarahkan ke kotak proyek, sebuah lapisan gelap (overlay) dengan judul proyek akan muncul.

.project-box .overlay {
    /* Posisi absolut untuk menutupi gambar */
    position: absolute; 
    background: rgba(0, 0, 0, 0.6);
    opacity: 0; /* Awalnya transparan */
    transition: opacity 0.4s ease;
}

.project-box:hover .overlay {
    opacity: 1; /* Muncul saat di-hover */
}

4. Logika JavaScript (script.js)

JavaScript digunakan untuk membuat efek ketik (typing effect) pada peran (role) di bawah nama Anda di halaman utama.

Cara Kerja Efek Ketik

  1. Menunggu DOM Siap: Skrip berjalan setelah seluruh halaman HTML dimuat, menggunakan DOMContentLoaded.
  2. Inisialisasi Variabel:
    • roles: Sebuah array yang berisi teks yang akan diketik (misal: "WEB DESIGNER").
    • roleIndex, charIndex: Penunjuk untuk teks dan karakter saat ini.
    • isDeleting: Sebuah flag (penanda) untuk mengetahui apakah skrip sedang dalam mode mengetik atau menghapus.
  3. Fungsi type(): Ini adalah fungsi utama yang berjalan berulang kali.
    • Jika isDeleting true: Skrip akan menghapus karakter satu per satu dari teks yang ditampilkan. Jika semua sudah terhapus, isDeleting diubah menjadi false dan skrip pindah ke role berikutnya.
    • Jika isDeleting false: Skrip akan menambahkan karakter satu per satu. Jika seluruh kata sudah tertulis, isDeleting diubah menjadi true dan ada jeda sejenak sebelum mulai menghapus.
  4. Rekursi dengan setTimeout: Fungsi type() memanggil dirinya sendiri setelah jeda waktu tertentu (typingSpeed atau deletingSpeed), menciptakan ilusi animasi mengetik.

5. Panduan Kustomisasi

Bagian ini memberikan langkah-langkah praktis untuk mengubah konten dan tampilan portofolio.

A. Mengubah Informasi Personal

  • Nama: Buka index.html dan ubah teks di dalam elemen <h1>.
  • Deskripsi Diri: Buka about.html dan ubah teks di dalam paragraf <p>.
  • Foto Profil: Ganti file images/portrait.png dengan foto Anda. Usahakan nama file tetap sama atau perbarui path di index.html.

B. Mengubah Peran (Typing Effect)

Buka file js/script.js. Cari baris ini dan ubah isinya:

const roles = ["WEB DESIGNER", "WEB DEVELOPER", "UI/UX DESIGNER"];

Anda bisa menambah, menghapus, atau mengubah teks di dalam array tersebut. Anda juga bisa mengatur kecepatan ketik dan hapus dengan mengubah nilai variabel typingSpeed dan deletingSpeed.

C. Mengupdate Proyek

Buka file projects.html dan temukan bagian .projects-grid.

<!-- Project 1 -->
<a href="#" class="project-box p1" target="_blank">
    <img src="images/project1.png" alt="Project 1 Screenshot">
    <div class="overlay">
        <h3>Framework Finder</h3>
    </div>
</a>
  • Link Proyek: Ubah nilai href="#" menjadi URL proyek Anda.
  • Gambar Proyek: Ganti file images/project1.png dengan screenshot proyek Anda, atau ubah path di src="...".
  • Nama Proyek: Ubah teks di dalam <h3>.

Ulangi untuk proyek lainnya.

D. Mengubah Link Sosial Media

Di setiap file HTML, temukan bagian <footer>. Ubah nilai href="#" pada setiap tag <a> dengan link profil sosial media Anda.

<div class="social-icons">
    <a href="https://twitter.com/username" aria-label="X"><i class="fa-brands fa-x-twitter"></i></a>
    <a href="https://github.com/username" aria-label="GitHub"><i class="fa-brands fa-github"></i></a>
    <!-- dan seterusnya -->
</div>

E. Mengubah Tema (Warna & Font)

Buka css/style.css dan ubah nilai variabel di bagian paling atas (:root).

/* Ganti dengan kode warna atau nama font yang Anda inginkan */
:root {
    --background-color: #F0F2F5; /* Contoh: Latar belakang abu-abu muda */
    --text-color: #2c3e50;       /* Contoh: Teks biru tua */
    --font-heading: 'Montserrat', sans-serif;
    --font-body: 'Lato', sans-serif;
}

Jangan lupa untuk mengimpor font baru dari Google Fonts di baris paling atas file CSS jika diperlukan.
